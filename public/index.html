DesignByMeg — Static Snapshot

This is a minimal placeholder index.html placed in public/ to let you deploy to Firebase Hosting. Replace this file (and add the rest of your static HTML/CSS/JS files) inside the public/ folder before deploying.

If your original site is PHP/WordPress you must export it to static HTML first (or host PHP elsewhere). Firebase Hosting serves static files only.

Next steps

  1. Install Firebase CLI and log in (see README_FIREBASE.md).
  2. Place your static site files in public/ (index.html, assets/ ...).
  3. Run firebase deploy --only hosting to publish.
